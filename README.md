## Objective Develop a **console-based file explorer** in **C++** that interfaces with the Linux operating system to manage files and directories. 

## Features
- List and explore files/directories
- Navigate through directories
- Create, move, copy, and delete files
- Search for files recursively
- View and modify file permissions
- User-friendly command-line interface
  
- ## How to Run bash
- Compile g++ -std=c++17 file_explorer.cpp -o file_explorer
- ./explorer
